# Portafolio Profesional | Ingeniería en Sistemas Computacionales y Electrónicos

## Descripción General
Este repositorio contiene el código fuente y la estructura del portafolio técnico de Ulises Daniel Vargas Nedzelsky, estudiante de quinto cuatrimestre de la carrera de Ingeniería en Sistemas Computacionales y Electrónicos en la Universidad del Noreste (UNE). 

El sitio web funciona como un documento técnico que expone el desarrollo de soluciones integrales, abarcando desde la arquitectura de bases de datos relacionales y el desarrollo de software, hasta la implementación de sistemas embebidos y la creación de prototipos de hardware.

## Proyectos Documentados

El portafolio detalla el diseño, ejecución y resultados de las siguientes iniciativas:

* **Sistema WING (Automatización de Pagos):** Plataforma para el registro de información e interfaz de pagos. Su arquitectura demuestra el dominio avanzado de SQL Server (implementación de procedimientos almacenados, funciones, *triggers* y planes de respaldo), integrado con interfaces web mediante HTML/JavaScript y sincronización de datos a través de la API de Google Sheets.
* **Prótesis Robótica con EMG (LSM):** Prototipo electromecánico diseñado para interpretar señales electromiográficas (EMG) y reproducir gesticulaciones precisas del Lenguaje de Señas Mexicano (LSM). El proyecto evidencia competencias en instrumentación biomédica, uso de microcontroladores (Arduino), software de simulación y calibración de servomotores MG90S.

## Stack Tecnológico

Los proyectos exhibidos en este portafolio demuestran competencias operativas en las siguientes áreas:

* **Software y Web:** JavaScript, HTML, CSS.
* **Gestión de Datos:** SQL Server (Transact-SQL).
* **Hardware y Sistemas Embebidos:** C++ (Entorno Arduino), modelado de circuitos (Tinkercad), compuertas lógicas e instrumentación analógica.
